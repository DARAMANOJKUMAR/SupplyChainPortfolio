Evaluate and choose suppliers using scoring, ranking, and optimization techniques

Linear Point Method

Steps: Assign points to each criterion, then sum across all criteria for each supplier.

Use Case: Ideal for quick, consistent evaluations when weights are equal or predefined.

Example: Used by a mid-sized retail firm to rank raw material vendors based on availability, price, and distance.

Ranking Method (Borda Count)

Steps: Rank suppliers on each criterion, convert ranks into points, and sum totals.

Use Case: Good when relative performance matters more than absolute numbers.

Example: FMCG firm used Borda ranking to select best packaging vendor based on logistics, speed, and cost.

Goal Programming (Vendor Allocation)

Steps: Formulate constraints (e.g., demand, budget) and minimize deviations from goals using LP.

Use Case: Effective for allocating order quantities when multiple suppliers are required.

Example: A manufacturing unit used this method to assign bulk orders across 3 suppliers while minimizing cost.
