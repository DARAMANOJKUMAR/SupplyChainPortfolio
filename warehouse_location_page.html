Factor Rating Method

Steps: Assign weights and scores to location factors and compute weighted totals.

Use Case: Helpful when qualitative and quantitative criteria must be balanced.

Example: A retail chain evaluates new warehouse cities based on land cost, labor, and distance.

GitHub Link

Break-even Analysis

Steps: Compare fixed and variable costs across sites to find the lowest total cost location.

Use Case: Ideal for selecting locations where cost changes with volume shipped.

Example: A beverage distributor selects warehouse sites based on cost break-even charts.

GitHub Link

Centre of Gravity Method

Steps: Use weighted average of coordinates to find geographically central location.

Use Case: Best when minimizing transport distances to multiple customer zones.

Example: An e-commerce company chooses its hub location based on customer density map.

GitHub Link

Total Cost Optimization (MILP)

Steps: Formulate and solve a mixed integer linear programming model to minimize total cost.

Use Case: Used when location decisions involve cost, distance, and capacity simultaneously.

Example: A national logistics company uses MILP to optimize warehouse openings and assignments.
