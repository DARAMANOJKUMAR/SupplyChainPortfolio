Compare alternatives using structured multi-criteria decision-making models

AHP (Analytic Hierarchy Process)

Steps: Build a decision hierarchy, do pairwise comparisons, and compute priority weights.

Use Case: Best for structured decisions with both tangible and intangible criteria.

Example: A steel firm selects software vendors based on cost, support, and flexibility.

GitHub Link

TOPSIS

Steps: Normalize data, identify ideal/negative ideal, compute closeness to ideal solution.

Use Case: Suitable for ranking suppliers based on multiple weighted criteria.

Example: A procurement team ranks transport agencies on punctuality, cost, and coverage.

GitHub Link

VIKOR

Steps: Normalize performance scores, calculate utility and regret measures, derive compromise ranking.

Use Case: When decisions require balance between group utility and individual regret.

Example: A logistics firm uses VIKOR to select the best packaging material supplier under multiple trade-offs.
